# Moodle AI Solver 🎓

This is a Solver for the Moodle of Basis Schools, and Basis Schools ONLY.

**Status:** ✅ Working as of April 27th of 2026

---

## ✨ Features
* **One-Click Extraction:** Quickly grab question text from Moodle pages.
* **Universal AI Support:** Compatible with Gemini, ChatGPT, Claude, and more.
* **Seamless Integration:** Runs directly in your browser via Tampermonkey.

## 🚀 Installation

This tool is a **Userscript**. To use it, follow these steps:

1.  **Install Tampermonkey:** Download the extension for [Chrome](https://google.com/search?q=tampermonkey+chrome), [Firefox](https://google.com/search?q=tampermonkey+firefox), or [Edge](https://google.com/search?q=tampermonkey+edge).
2.  **Create New Script:** Click the Tampermonkey icon in your browser and select "Create a new script."
3.  **Paste the Code:** Delete any template code and paste the content of `Moodle AI.user.js` (found in this repo).
4.  **Save:** Press `Ctrl+S` or File > Save.
5.  If you already have Tampermonkey installed, click [here](https://raw.githubusercontent.com/vhuicatnogr/Moodle-AI/main/Moodle%20AI.user.js)

## 🛠️ How to Use

1.  Navigate to your Moodle quiz.
2.  The solver interface will appear.
3.  **Extract:** Copy the question content provided by the script.
4.  **Solve:** Paste it into your preferred AI.
5.  **Inject:** Paste the AI's response back into the script's input field to automate the Moodle form.
